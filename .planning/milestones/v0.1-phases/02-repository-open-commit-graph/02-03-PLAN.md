    **src-tauri/src/git/graph.rs** — Implement:

    `pub fn walk_commits(repo: &git2::Repository, offset: usize, limit: usize) -> Result<Vec<crate::git::types::GraphCommit>, crate::error::TrunkError>`:

    Step 1 — Build ref map: call `repository::build_ref_map(repo)`

    Step 2 — Revwalk to collect all OIDs:
    - `let mut revwalk = repo.revwalk()?;`
    - Push refs separately to avoid glob depth issue: `revwalk.push_glob("refs/heads")?; revwalk.push_glob("refs/remotes")?; revwalk.push_glob("refs/tags")?;`
    - Set sort: `revwalk.set_sorting(git2::Sort::TOPOLOGICAL | git2::Sort::TIME)?;`
    - Collect: `let oids: Vec<git2::Oid> = revwalk.collect::<Result<Vec<_>, _>>()?;`

    Step 3 — Apply offset/limit to get the slice of OIDs to process for output:
    - `let page_oids = &oids[offset.min(oids.len())..(offset + limit).min(oids.len())];`

    Step 4 — Lane assignment algorithm (single pass over ALL oids to maintain continuity across batches, but only output the page slice):
    - active_lanes: `Vec<Option<git2::Oid>>` — index = column, None = free lane
    - pending_parents: `HashMap<git2::Oid, usize>` — oid → reserved column (assigned by a prior child commit)
    - per_oid_column: `HashMap<git2::Oid, (usize, Vec<GraphEdge>)>` — store column + edges for every OID

    For each oid in the full oids list:
    1. Find column: if pending_parents contains this oid → take that column; else find first free slot in active_lanes (or push new)
    2. Get commit from repo: `repo.find_commit(oid)?`
    3. is_merge: commit.parent_count() >= 2
    4. Edges for this row:
       - For each column slot that is Some(other_oid) and other_oid != this_oid → emit Straight edge (from=col, to=col, color=col)
       - Remove this_oid from active_lanes (set slot to None — it's been consumed)
    5. For each parent oid (commit.parents()):
       - First parent: reuse the current commit's column (continuity); put parent_oid in active_lanes[column]
       - Subsequent parents: if already in pending_parents use that column; else find first free slot
       - Emit ForkLeft or ForkRight edge from commit's column to parent's column (ForkLeft if parent col < commit col, ForkRight if parent col > commit col, Straight if same)
       - Add to pending_parents[parent_oid] = parent_column
    6. Store (column, edges) in per_oid_column

    Step 5 — Build output Vec<GraphCommit> for page_oids only:
    - For each oid in page_oids:
      - commit = repo.find_commit(oid)?
      - (column, edges) = per_oid_column[&oid]
      - refs = ref_map.get(&oid).cloned().unwrap_or_default()
      - is_head = refs.iter().any(|r| r.is_head)
      - Build GraphCommit with all fields populated from commit data + column + edges + refs
    - Return Vec<GraphCommit>

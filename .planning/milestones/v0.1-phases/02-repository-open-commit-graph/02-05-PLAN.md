SVG lane coordinate system (from 02-RESEARCH.md Pattern 5):
- laneWidth = 12px per lane column
- rowHeight matches component row height (26px)
- cx(col) = col * laneWidth + laneWidth / 2
- cy = rowHeight / 2
- Straight edges: line from (cx(from), 0) to (cx(to), rowHeight)
- Fork/Merge edges: Bézier curve M cx(from) cy C cx(from) rowHeight, cx(to) 0, cx(to) rowHeight
- Lane color: var(--lane-{color_index % 8})
- Regular dot: r=4, fill=lane color, no stroke
- Merge dot: r=6, fill=lane color, stroke=var(--color-bg) strokeWidth=2
- SVG total width: (maxColumn + 1) * laneWidth; compute maxColumn from commit.edges max from/to column

    **src/components/LaneSvg.svelte** — per-row inline SVG lane:

    Props: `{ commit: GraphCommit; laneWidth?: number; rowHeight?: number }`
    Defaults: laneWidth = 12, rowHeight = 26

    Computed values:
    - `const cx = (col: number) => col * laneWidth + laneWidth / 2;`
    - `const cy = rowHeight / 2;`
    - `const laneColor = (idx: number) => \`var(--lane-\${idx % 8})\`;`
    - maxCol: max of all from_column and to_column across commit.edges, and commit.column — use this to compute SVG width
    - svgWidth: (maxCol + 1) * laneWidth (minimum laneWidth to avoid 0-width SVG)

    Render:
    ```svelte
    <svg width={svgWidth} height={rowHeight} style="overflow: visible; flex-shrink: 0;">
      <!-- Edges (drawn first, below the dot) -->
      {#each commit.edges as edge}
        {#if edge.edge_type === 'Straight'}
          <line
            x1={cx(edge.from_column)} y1={0}
            x2={cx(edge.to_column)}   y2={rowHeight}
            stroke={laneColor(edge.color_index)} stroke-width="2"
          />
        {:else}
          <!-- ForkLeft, ForkRight, MergeLeft, MergeRight use Bézier curve -->
          <path
            d={`M ${cx(edge.from_column)} ${cy} C ${cx(edge.from_column)} ${rowHeight}, ${cx(edge.to_column)} ${0}, ${cx(edge.to_column)} ${rowHeight}`}
            fill="none" stroke={laneColor(edge.color_index)} stroke-width="2"
          />
        {/if}
      {/each}

      <!-- Commit dot -->
      <circle
        cx={cx(commit.column)}
        cy={cy}
        r={commit.is_merge ? 6 : 4}
        fill={laneColor(commit.column % 8)}
        stroke={commit.is_merge ? 'var(--color-bg)' : 'none'}
        stroke-width={commit.is_merge ? 2 : 0}
      />
    </svg>
    ```

    Props: `{ repoPath: string }`

    State:
    - `let commits = $state<GraphCommit[]>([]);`
    - `let hasMore = $state(true);`
    - `let loading = $state(false);`
    - `let error = $state<string | null>(null);`
    - `let offset = $state(0);`
    - `const BATCH = 200;`
    - `const SKELETON_COUNT = 10; // rows to show while loading`

    loadMore function:
    ```typescript
    async function loadMore() {
      if (loading || !hasMore) return;
      loading = true;
      error = null;
      try {
        const batch = await safeInvoke<GraphCommit[]>('get_commit_graph', {
          path: repoPath,
          offset,
        });
        commits.push(...batch);   // Svelte 5: direct push is reactive
        offset += batch.length;
        if (batch.length < BATCH) hasMore = false;
      } catch (e) {
        const err = e as TrunkError;
        error = err.message ?? 'Failed to load commits';
      } finally {
        loading = false;
      }
    }
    ```

    Initial load via $effect: `$effect(() => { loadMore(); });`

    Template structure:
    - If initial load (commits.length === 0 && loading): render SKELETON_COUNT skeleton rows
    - If initial load error: render inline error banner with error message (styled with red border/background using Tailwind)
    - Otherwise: render SvelteVirtualList with items={commits}, onLoadMore={loadMore}, loadMoreThreshold={50}, hasMore={hasMore}
      - {#snippet renderItem(commit)} `<CommitRow {commit} />` {/snippet}
      - After the list: if loading && commits.length > 0, render 3 skeleton rows at bottom
      - After the list: if error && commits.length > 0, render error indicator + Retry button (onclick calls loadMore())

    Import: `import SvelteVirtualList from '@humanspeak/svelte-virtual-list';`

Established App.svelte $effect listener pattern (from StagingPanel.svelte):
```typescript
$effect(() => {
  let unlisten: (() => void) | undefined;
  listen<string>('repo-changed', (event) => {
    if (event.payload === repoPath) handleRefresh();
  }).then((fn) => { unlisten = fn; });
  return () => { unlisten?.(); };
});
```

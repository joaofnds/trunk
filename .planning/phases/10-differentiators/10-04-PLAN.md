In src/components/CommitRow.svelte:

1. **Connector line dimming for remote-only refs:** Add a derived `allRemoteOnly` check:
   ```ts
   const allRemoteOnly = $derived(
     commit.refs.length > 0 &&
     commit.refs.every(r => r.ref_type === 'RemoteBranch')
   );
   ```
   On the connector line div (lines 38-41), add `opacity: ${allRemoteOnly ? 0.5 : 1};` to the style string.

2. **Connector line overshoot fix:** The connector line currently starts at `left:0` and spans the full ref column. Instead, measure the actual pill container width using `bind:clientWidth`:
   - Add a `let refContainerWidth = $state(0);` variable.
   - On the ref column wrapper div (line 45), add `bind:clientWidth={refContainerWidth}`.
   - Change the connector line div to start AFTER the pills: set `left: {8 + refContainerWidth}px` (8 = px-2 left padding of the row) and `width: {columnWidths.ref - refContainerWidth + commit.column * 12 + 6}px`. This makes the line span from the right edge of the pill content to the commit dot center.
   - Keep the `z-index: 0` and `pointer-events-none` on the connector.

3. **WIP dotted line hover fix:** The HEAD row's `hover:bg-[var(--color-surface)]` (opaque) covers the WIP SVG overflow. Fix by giving the graph column div (line 50) a `position: relative` and `z-index: 1` so the SVG overflow from the WIP row above paints correctly. Change line 50 to add `relative z-[1]` classes to the graph column div.

4. **Data row column dividers:** Add a subtle right border to every column div EXCEPT the last (SHA). On the ref, graph, message, author, and date column divs, add inline style `border-right: 1px solid var(--color-border);`. The SHA column (last) gets no right border.

5. **Cell padding:** Add `px-1` (or equivalent `padding: 0 4px`) to text-bearing column cells: ref, message, author, date, SHA. The graph column does NOT need text padding (it contains SVG). Also add `pl-1` to the ref column container so pills have breathing room from the left edge.

In src/components/RefPill.svelte:

6. **Overflow count styling:** Change the overflow `+N` span (lines 44-49) from plain muted text to a small pill. Replace the current span with:
   ```svelte
   <span
     class="inline-flex items-center rounded-full px-1 text-[10px] leading-4 whitespace-nowrap font-medium ml-1 cursor-default"
     style="background: var(--color-surface); color: var(--color-text-muted); border: 1px solid var(--color-border);"
     title={refs.slice(1).map((r) => r.short_name).join(', ')}
   >
     +{refs.length - 1}
   </span>
   ```
   This gives it a rounded background, border, and padding so it is readable against the connector line.
